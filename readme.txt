Masters of Arts: ML Challenge распознавание вида произведения по фотографии
Скрипты:
1. image_resizing.py
1.1. Функционал - изменение размеров картинок по тербуемому формату для предобученной нейросети
1.2. Входные данные - оригинальные рисунки
1.3. Выходные данные - рисурки с измененным размером (299 х 299) пикселей

2. image_distribution.py
2.1. Функционал - создает файловую структуру (распределяет тренеровочные файлы по папкам, в соответствии с их классом)
для использования в пайплайне обучения
2.2. Входные данные - папка с файлами с измененными размерами
2.3. Выходные данные - папка с вложенными папками с названиями, которых являются номерами классов, заполненные картинками

3. train_model_zip.ipynb
3.1. Функционал - обучает загруженную нейросеть архитектуры VGG16 с весами "imagenet", замароженными CNN слоями и замененным 
слоем классификатора
3.2. Входные данные - zip архив со структурой папок для пайплайна загрузки keras.image_dataset_from_directory
zip используется для загрузки данных сразу полностью в оперативку colab для ускорения обучения - экономия на передаче данных
3.3. Выходные данные - модель, обученная на входных данных

4. fine_tune_train_model_zip.ipynb
4.1. Функционал - обучает загруженную нейросеть архитектуры Xception с весами "imagenet", замароженными CNN слоями и замененным 
слоем классификатора, далее размораживаем слои и дообучаем сеть полностью
4.2. Входные данные - zip архив со структурой папок для пайплайна загрузки keras.image_dataset_from_directory
zip используется для загрузки данных сразу полностью в оперативку colab для ускорения обучения - экономия на передаче данных
4.3. Выходные данные - модель, обученная на входных данных

5. test_model.py
5.1. Функционал - предсказание меток классификатора для тестовых данных и формирование отчета
5.2. Входные данные - тестовые картинки
5.3. Выходные данные - отчет
